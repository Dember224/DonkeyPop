# DonkeyPop

Donkeypop is a project for collecting social media data on Democratic, Democratic caucusing, left leaning, or leftist candidates.
Once we've got the data we can do a little analysis to figure out how these folks can do it better.

## Initialization

CD into the donkey pop directory. Go ahead and run ```make initialize```

## Running DonkeyPop

Run ```make start```

## Stopping DonkeyPop

Run ```make stop```
